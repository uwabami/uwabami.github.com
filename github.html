主に自分が使うために作成したソフトウェア、 ひきとったソフトウェア、 プレゼン用のテーマなどです. 公開できるモノについては github にどんどん置いています. パッチウェルカム.

RDtool: formatter for RD

RD は Ruby における POD(Plain Old Documentation) のようなものです. RDtool は RD のフォーマッタであり, RD を HTML や roff などに変換できます. 以前は Moonwolf さんがリリースマネジメントをされていましたが, 2011年 5 月に引き継いで, gem 化したりイロイロ弄っています.

  • gem で取得できます: RubyGems.org:rdtool
  • 開発は GitHub でやってます: Github: rdtool
  • ご意見, ご要望など, メール下さるか Github の Issues にお願いします.

Rail: Replace Agent-string Internal Library

しまだみつひろさんが作成された rail を勝手に引き取って公開/気がついた時にアップデートしています. お気付きの点がありましたら, ご連絡下さい.

  • 取得は github からどうぞ: Github: rail
  • ご意見, ご要望など, メール下さるか Github の Issues にお願いします.
  • Debian/Ubuntu ユーザの方は, 公式パッケージ にしましたので apt-get で install できます.
     % sudo apt-get install rail
