* RDってなに？
  {{{travis(uwabami/rdtool)}}}
  {{{ghribbon(uwabami/rdtool)}}}

  RDはRubyにおけるPOD(Plain Old Documentation)のようなものです。
  RDtoolはRDのフォーマッタであり、RDをHTMLなどに変換できます。
* Download
** stable 版: Gem もしくは tar.gz
   - [[https://github.com/uwabami/rdtool/tarball/v0.6.38][github tarball]]
   - [[http://rubygems.org/gems/rdtool]]
** HEAD: GitHub: [[https://github.com/uwabami/rdtool/][uwabami/rdtool]]
   - [[https://github.com/uwabami/rdtool/zipball/master][HEAD zip]]
   - [[https://github.com/uwabami/rdtool/tarball/master][HEAD tarball]]
* インストール方法
  Gem を使う場合:
  #+BEGIN_EXAMPLE
    % gem install rdtool
  #+END_EXAMPLE

  Gem を使わない場合:
  #+BEGIN_EXAMPLE
    % tar xvzf rdtool-(version).tar.gz
    % cd rdtool-(version)
    % ruby setup.rb    # or sudo ruby setup.rb
  #+END_EXAMPLE
