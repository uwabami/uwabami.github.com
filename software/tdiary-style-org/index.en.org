* What's this?
  {{{travis(uwabami/tdiary-style-org)}}}
  {{{ghribbon(uwabami/tdiary-style-org)}}}


  The =TDiary::Style::Org= is provide [[http://orgmode.org/][Org-mode]] style for [[http://www.tdiary.org/][tDiary]].
  Convert Org to HTML by  [[https://github.com/wallyqs/org-ruby][wallyqs/org-ruby: An Org mode parser written in Ruby]].
  #+BEGIN_EXPORT html
  <div class="col-md-6 col-sm-12">
  <img src="./src.png" class="img-responsive">
  </div>
  <div class="col-md-6 col-sm-12">
  <img src="./result.png" class="img-responsive">
  </div>
  #+END_EXPORT
* Installation
  Add this line to your application's Gemfile.local:
  #+BEGIN_SRC ruby
    gem 'tdiary-style-org'
  #+END_SRC
  And then execute:
  #+BEGIN_EXAMPLE
    % bundle install
  #+END_EXAMPLE
  Or install it yourself as:
  #+BEGIN_EXAMPLE
    % gem install tdiary-style-org
  #+END_EXAMPLE
* Usage
  If you want to use this style, add @style into tdiary.conf below:
  #+BEGIN_SRC ruby
    @style = 'Org'
  #+END_SRC
* Syntax
  Basically, samve as  [[http://orgmode.org/][Org-mode]].
  - First Heading =*= is converted to =<h3>=
  - Org-mode =TAG= is converted to CATEGORY of tDiary
  - plugin syntax: =((%plugin, 'val'%))=
