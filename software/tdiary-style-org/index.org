* これは何?
  {{{travis(uwabami/tdiary-style-org)}}}
  {{{ghribbon(uwabami/tdiary-style-org)}}}

  TDiary::Style::Org は
  [[http://orgmode.org/][Org mode for Emacs – Your Life in Plain Text]] の syntax で
  [[http://www.tdiary.org/][tDiary.org]] の記事を書くためのスタイルを提供します．

  Org から HTML の変換には [[https://github.com/wallyqs/org-ruby][wallyqs/org-ruby: An Org mode parser written in Ruby]] を用いています．
  また，ソースコードのシンタックスハイライトには [[https://github.com/tmm1/pygments.rb][tmm1/pygments.rb: pygments syntax highlighting in ruby]]
  を使っています．
  #+BEGIN_EXPORT html
  <div class="col-md-6 col-sm-12">
  <img src="./src.png" class="img-responsive">
  </div>
  <div class="col-md-6 col-sm-12">
  <img src="./result.png" class="img-responsive">
  </div>
  #+END_EXPORT
* インストール
  tDiary の =Gemfile.local= に
  #+BEGIN_SRC ruby
    gem 'tdiary-style-org'
  #+END_SRC
  を追加して，
  #+BEGIN_EXAMPLE
    % bundle install
  #+END_EXAMPLE
  とするか，
  #+BEGIN_EXAMPLE
    % gem install tdiary-style-org
  #+END_EXAMPLE
  として下さい．
* 使い方
  =tdiary.conf= の =@style= で以下の用に指定します．
   #+BEGIN_SRC ruby
@style = 'Org'
   #+END_SRC
* 文法
  基本的には通常の Org-mode の文法です．
  ただし，
  - Org-mode の TAG が tDiary の CATEGORY になります．
  - plugin 記法は今のところ =((%plugin, 'val'%))= となっています．
    もっと良い(Org らしい)記法があればご提案頂けると幸いです．
