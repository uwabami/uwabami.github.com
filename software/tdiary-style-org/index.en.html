What's this?

Fork me on GitHub

The TDiary::Style::Org is provide Org-mode style for tDiary. Convert Org to HTML by wallyqs/org-ruby: An Org mode parser written in Ruby.

Installation

Add this line to your application's Gemfile.local: 

    gem 'tdiary-style-org'

And then execute: 

    % bundle install

Or install it yourself as: 

    % gem install tdiary-style-org

Usage

If you want to use this style, add @style into tdiary.conf below: 

    @style = 'Org'

Syntax

Basically, samve as Org-mode.

  • First Heading * is converted to <h3>
  • Org-mode TAG is converted to CATEGORY of tDiary
  • plugin syntax: ((%plugin, 'val'%))
