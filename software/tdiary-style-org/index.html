_これは何?

{{{travis(uwabami/tdiary-style-org)}}} {{{ghribbon(uwabami/tdiary-style-org)}}}

TDiary::Style::Org は Org mode for Emacs – Your Life in Plain Text の syntax で tDiary.org の記事を書くためのスタイルを提供します．

Org から HTML の変換には wallyqs/org-ruby: An Org mode parser written in Ruby を用いています． また，ソースコードのシンタックスハイライトには tmm1/pygments.rb: pygments syntax highlighting in ruby を使っています．

_インストール

tDiary の Gemfile.local に

gem 'tdiary-style-org'

を追加して，

% bundle install

とするか，

% gem install tdiary-style-org

として下さい．

_使い方

tdiary.conf の @style で以下の用に指定します．

@style = 'Org'

_文法

基本的には通常の Org-mode の文法です． ただし，

  • Org-mode の TAG が tDiary の CATEGORY になります．
  • plugin 記法は今のところ ((%plugin, 'val'%)) となっています． もっと良い(Org らしい)記法があればご提案頂けると幸いです．
