** [[http://uwabami.github.io/rdtool/][RDtool: formatter for RD]]
   RD は Ruby における POD(Plain Old Documentation) のようなものです.
   RDtool は RD のフォーマッタであり, RD を HTML や roff などに変換できます.
   以前は Moonwolf さんがリリースマネジメントをされていましたが,
   2011年 5 月に引き継いで, gem 化したりイロイロ弄っています.
   - gem で取得できます: [[https://rubygems.org/gems/rdtool][RubyGems.org:rdtool]]
   - 開発は GitHub でやってます: [[https://github.com/uwabami/rdtool][Github: rdtool]]
   - ご意見, ご要望など, メール下さるか Github の [[https://github.com/uwabami/rdtool/issues][Issues]] にお願いします.
** [[http://uwabami.github.io/rail/][Rail: Replace Agent-string Internal Library]]
   しまだみつひろさんが作成された rail を勝手に引き取って公開/気がついた時にアップデートしています.
   お気付きの点がありましたら, ご連絡下さい.
   - 取得は github からどうぞ: [[https://github.com/uwabami/rail][Github: rail]]
   - ご意見, ご要望など, メール下さるか Github の [[https://github.com/uwabami/rail/issues][Issues]] にお願いします.
   - Debian/Ubuntu ユーザの方は, [[https://tracker.debian.org/pkg/rail][公式パッケージ]] にしましたので apt-get で install できます.
   #+BEGIN_EXAMPLE
     % sudo apt-get install rail
   #+END_EXAMPLE
