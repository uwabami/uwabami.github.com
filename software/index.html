RDtool: formatter for RD

RD は Ruby における POD(Plain Old Documentation) のようなものです. RDtool は RD のフォーマッタであり, RD を HTML や roff などに変換できます. 以前は Moonwolf さんがリリースマネジメントをされていましたが, 2011年 5 月に引き継いで, gem 化したりイロイロ弄っています.

しまだみつひろさんが作成された rail を勝手に引き取って公開/気がついた時にアップデートしています. お気付きの点がありましたら, ご連絡下さい.

TDiary::Style::Org

TDiary::Style::Org は Org mode for Emacs – Your Life in Plain Text の syntax で tDiary.org の記事を書くためのスタイルを提供します．
