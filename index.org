* What's this?

主に自分が使うために作成したソフトウェアやプレゼン用のテーマなどです.
公開できるモノについては github にどんどん置いています. パッチウェルカム.

* [[http://uwabami.github.com/rdtool/][RDtool: formatter for RD.]]

RD は Ruby における POD(Plain Old Documentation) のようなものです.
RDtool は RD のフォーマッタであり, RD を HTML や roff などに変換できます.
以前は Moonwolf さんがリリースマネジメントをされていましたが,
2011年 5 月に引き継いで, gem 化したりイロイロ弄っています.

 - gem で取得できます: [[https://rubygems.org/gems/rdtool][RubyGems.org:rdtool]]
 - 開発は GitHub でやってます.: [[https://github.com/uwabami/rdtool][Github: rdtool]]
 - ご意見, ご要望など, メール下さるか Github の [[https://github.com/uwabami/rdtool/issues?sort%3Dcreated&amp%3Bdirection%3Ddesc&amp%3Bstate%3Dopen][Issues]] にお願いします.

* [[http://uwabami.github.com/rail/][Rail: Replace Agent-string Internal Library]]

しまだみつひろさんが作成された rail を勝手に引き取って公開/気がついた時に
アップデートしています. お気付きの点がありましたら, ご連絡下さい.

 - 取得は github からどうぞ: [[https://github.com/uwabami/rail][Github: rail]]
 - ご意見, ご要望など, メール下さるか Github の [[https://github.com/uwabami/rdtool/issues?sort%3Dcreated&amp%3Bdirection%3Ddesc&amp%3Bstate%3Dopen][Issues]] にお願いします.
 - Debian/Ubuntu ユーザの方は, 公式パッケージにしましたので  apt-get で install できます.
